# Project Assets

This folder stores non-runtime project artifacts (not app-bundled resources).

## Structure

- `achievements/`: screenshots and evidence from completed tasks/milestones.
- `design/`: source design assets (icon source files, diagrams, etc.).

## Naming

- Achievement screenshots: `YYYY-MM-DD_<task-or-feature-slug>.png`
- Keep milestone grouping under `achievements/<milestone>/`.
